# Overview

A Game Project built in C++ using the SDL Library for the course 'CS224: Object Oriented Programming' at Habib University.

# Tasks

idk

## How to Run

-   Any IDE that supports C++.
-   Install SDL Library.
-   Run the following command to compile.

```
g++ *.cpp
```
